# Debug Checkpoints

> Snapshots of progress at key milestones. Use to rollback if needed.

---

## Checkpoint Format

Each checkpoint captures:
- **Files created/modified** in that phase
- **Decisions made** and rationale
- **State** that can be restored

---

## Checkpoints

### CP-000: Initial State
- **Time:** Session start
- **Files:** Only CLAUDE.md and .debug/ structure
- **State:** Clean slate, awaiting requirements
